# EcoMission-Pro
## Este es el tipo de mi proyecto:
> Una aplicación web interactiva tipo juego educativo que motiva a los usuarios a realizar acciones ecológicas, ganar puntos, subir de nivel y visualizar su impacto ambiental.

---

## Las bibliotecas que voy a utilizar:
- HTML5
- CSS3
- JavaScript (Vanilla)
- LocalStorage (para guardar progreso del usuario en el navegador)
